# Security Policy

## Supported Versions

GigaScript has two file types: **.g** and **.gsx**. Each have their own version as they use different versions of the GigaScript interpreter.

### .g
| Version | Status             |
| ------- | ------------------ |
| v2.0.0  | NOT RELEASED       |
| v1.0.0  | RELEASED           |

### .gsx
| Version | Status             |
| ------- | ------------------ |
| v2.0.0  | NOT RELEASED       |
| v1.0.0  | RELEASED           |
